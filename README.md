# React Timesheet Application #

## Getting started ##

Install dependencies:
`npm install`

Run the API:
`npm run api:start`

Start the webpack dev server:
`npm start`

View the application:
`Browse to localhost:4500`

## Technology ##

### Web app
React v15.4
React-router v4.0
Redux v3.6
Material-UI v0.16
